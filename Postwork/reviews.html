Sobre HTML, CSS y Sass

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para estructurar el contenido en la web. Define la estructura de una página mediante etiquetas que representan elementos como encabezados, párrafos, enlaces, imágenes, entre otros.

CSS (Cascading Style Sheets) se utiliza para dar estilo y mejorar la presentación de las páginas HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales de la página.

Sass es una extensión de CSS que agrega características poderosas, como variables, anidamiento y funciones. Facilita la escritura de estilos más eficientes y mantenibles.
